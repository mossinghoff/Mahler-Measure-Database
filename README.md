# Mahler-Measure-Database
<p>This site houses a number of lists concerning the Mahler measure of polynomials with integer coefficients. These lists are connected to the study of Lehmer's problem concerning polynomials with small Mahler measure, Mahler's problem regarding Littlewood polynomials with large Mahler measure, as well as questions regarding small Salem numbers, and bivariate polynomials with small Mahler measure.</p>

<p>A brief synopsis of each file appears here.</p>

| File | Description |
| :---- | :---- |
| AllSmallMeasures_200.txt        | All known primitive, irreducible, noncyclotomic polynomials with Mahler measure < 1.3 and degree ≤ 200. |
| BestByDegree_200.txt            | Smallest known Mahler measure > 1 for primitive irreducible polynomials of even degree ≤ to 200. |
| BestByHeight.txt                | Smallest known Mahler measure > 1 for primitive irreducible polynomials with fixed height. |
| BestByRootCount_200.txt         | Smallest known Mahler measure > 1 for primitive irreducible polynomials with a fixed number of roots outside the unit circle. |
| BestMeasures.txt                | All known Mahler measures in (1, 1.25) for primitive irreducible polynomials. |
| LargeLittlewood.txt             | The largest Mahler measures for Littlewood polynomials of each fixed degree ≤ 30. |
| LargeSkewSymmLittlewood.txt     | The largest Mahler measures for skew-symmetric Littlewood polynomials of each fixed even degree ≤ 60. |
| Littlewood_All72.txt            | All reciprocal Littlewood polynomials with degree ≤ 72 and Mahler measure in (1, 5/3). |
| Littlewood_Top40.txt            | The 40 smallest known Mahler measures > 1 of Littlewood polynomials. |
| SalemList.txt                   | All known Salem numbers < 49/37 = 1.3243... . |
| SmallBivariate.txt              | All known Mahler measures in (1, 1.37) of bivariate integer polynomials. |
| SummaryReport_200.txt           | Counts of primitive, irreducible, noncyclotomic polynomials with Mahler measure < 1.3 and degree ≤ 200, broken down by number of roots outside the unit circle. |
